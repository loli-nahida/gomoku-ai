# 🧠 五子棋 AI - 深度学习人机对战

基于 **神经网络 + 蒙特卡洛树搜索 (MCTS)** 的网页版五子棋人机对战游戏，参考 AlphaZero 架构设计。

## 特性

- **神经网络评估**：6层残差网络，策略网络 + 价值网络双头输出
- **MCTS 搜索**：PUCT 选择策略，支持从 200 到 5000 次模拟的多级难度
- **启发式回退**：关闭 NN 时使用基于棋型威胁评估的启发式策略
- **自我对弈训练**：支持浏览器端在线自我对弈训练
- **纯前端实现**：无需后端，零依赖，浏览器直接运行

## 架构

```
Input (15×15×6 特征平面)
    ↓
Conv 3×3 (64 filters) + BN + ReLU
    ↓
Residual Block ×4 (Conv→BN→ReLU→Conv→BN + Skip)
    ↓
┌──────────┬──────────┐
│ Policy Head │ Value Head │
│ Conv→FC→Softmax │ Conv→FC→FC→Tanh │
│ → 225 概率 │ → [-1,1] 估值 │
└──────────┴──────────┘
```

### 输入特征平面 (6通道)

| 通道 | 含义 |
|------|------|
| 0 | 当前玩家棋子 |
| 1 | 对手棋子 |
| 2 | 最后落子位置 |
| 3 | 倒数第二步位置 |
| 4 | 威胁程度热力图 |
| 5 | 合法落子位置 |

## 操作说明

1. **新对局**：点击「新对局」按钮或按 `N` 键
2. **悔棋**：点击「悔棋」或按 `Ctrl+Z`
3. **难度调整**：左侧面板选择 AI 难度
4. **执棋选择**：可选择执黑（先手）或执白（后手）
5. **NN 开关**：可关闭神经网络评估，使用纯启发式 AI

## 技术细节

- **神经网络**：纯 JavaScript 实现的前向推理，He 初始化权重，BatchNorm，ReLU 激活
- **MCTS**：PUCT 公式平衡探索与利用，Dirichlet 噪声增加探索性
- **启发式评估**：五连、活四、冲四、活三等棋型识别，攻防兼顾
- **异步 AI**：长时间搜索自动分块执行，不阻塞 UI
